# Quality/Csat ID Hub
Central de links e recursos do pack Quality/Csat ID · Nubank.

## 🔗 Acesso
**[alanclovis.github.io/quality-id-hub](https://alanclovis.github.io/quality-id-hub)**

## Como usar
- Abra o link acima no browser
- Use a barra de busca ou pressione `/` para encontrar links rapidamente
- Passe o mouse sobre um card para ver a descrição

## Como editar os links
Apenas pessoas autorizadas podem editar. Para solicitar acesso, fale com Alan no Slack.

Quem tiver acesso:
1. Clique em **Editar** no canto superior direito
2. Clique no ✏️ do card que deseja alterar
3. Clique em **Salvar** para sincronizar para todos

## Tecnologia
HTML/JS estático hospedado no GitHub Pages.
Dados salvos em GitHub Gist (`quality-hub-data.json`). Sem backend, sem banco de dados.

**Dimensionamento:** grade semanal conectada à planilha Google Sheets via Apps Script (`gas/` + ponte iframe). Ver [docs/RUNBOOK-Dimensionamento.md](docs/RUNBOOK-Dimensionamento.md).
